# Primal Intelligence Workbook

So I read this book, *Primal Intelligence* by Angus Fletcher, and it basically told me my brain is way smarter than I've been giving it credit for, it just thinks in stories instead of spreadsheets. Cool! Typically though, I suck at actually implementing things I read in personal development books. This one has stuck with me and I wanted to make sure I actually implemented it.

So I built this.

It's a single HTML file. There's no external communication so no security concerns, entirely local to you. You download it, you open it, you use it. Your data is in your browser. It will not phone home. It does not know you exist. It is just a file (crazy I know).

---

## What's in it

- **Self-Assessment Quiz**: The actual 20-question diagnostic from the book, scored properly across your four primal powers: Intuition, Imagination, Emotion, and Commonsense
- **12-Chapter Workbook**: Reflections and exercises for every chapter, so you're actually working through it (I used a local LLM to generate a lot of the in-app text, so it sounds a little AIey, but good news no data centers were utilized, just my 4080 and a significant amount of heat in my office)
- **Story Builder**: A structured space to map your actual life narrative using the book's framework
- **Daily Practice**: A journal for building the habit 
- **Progress Tracking**: A radar chart, streaks, timeline, the works

## How to use it

1. Clone or download the repo, star it because ur cool
2. Open 'primal-workbook.html'
3. That's it

Your progress saves automatically to your browser's local storage. Use the Export button in the sidebar to back it up as a JSON file. Use Import to bring it back. Works across devices if you move the file and import your backup.
